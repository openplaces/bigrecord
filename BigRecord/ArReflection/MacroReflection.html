Overview

Abstract base class for AggregateReflection and AssociationReflection that describes the interface available for both of those classes. Objects of AggregateReflection and AssociationReflection are returned by the Reflection::ClassMethods.

  • - (Object) macro Returns the name of the macro, so it would return :composed_of for "composed_of :balance, :class_name => ‘Money’" or :has_many for "has_many :clients".

  • - (Object) options Returns the hash of options used for the macro, so it would return { :class_name => "Money" } for "composed_of :balance, :class_name => ‘Money’" or {} for "has_many :clients".
